      TENTANG APLIKASI

      SpinYuk Desktop

      SpinYuk Desktop adalah aplikasi spinner interaktif berbasis desktop yang dirancang untuk membantu pengguna memilih keputusan secara cepat, membagi kelompok secara acak, serta menjalankan simulasi spin bersama melalui mode Community Spin.

      Electron HTML CSS JavaScript Local Storage
      Mata Kuliah Pengembangan Aplikasi Berbasis Platform
      Platform Desktop Application
      Mode Aplikasi Quick, Group, dan Community Spin
      Tujuan Membantu pengambilan keputusan secara acak dan interaktif

      Fitur Utama

      Quick Spin

      Mode spinner utama untuk memilih satu hasil dari daftar pilihan.

      Group Spin

      Mode pembagian nama ke dalam beberapa kelompok secara bertahap.

      Community Spin

      Mode simulasi room bersama dengan room code dan password sederhana.

      Desktop Feature

      Mendukung system tray, shortcut, import file, export, drag and drop, dan focus mode.

      HOW TO USE

      Panduan Penggunaan SpinYuk

      Ikuti alur berikut untuk menggunakan fitur utama aplikasi.

      Pilih Mode

      Pilih Quick Spin, Group Spin, atau Community Spin melalui sidebar.

      Masukkan Data

      Tambahkan pilihan, daftar nama, atau room code sesuai mode yang digunakan.

      Mulai Spin

      Klik tombol spin untuk memutar roda dan menentukan hasil secara acak.

      Lihat Hasil

      Hasil spin akan muncul pada card hasil dan tersimpan ke riwayat.

      Export Data

      Simpan riwayat atau hasil kelompok ke file lokal sebagai dokumentasi.
